Tipe Data Dasar
1. Variabel dan Konversi Tipe
- Deklarasikan sebuah variabel string `num` dengan nilai "42" dan ubah menjadi angka.  
- Tambahkan `8` sehingga hasilnya menjadi `50`. (Konversi string ke angka dapat menggunakan `parseInt`)  
- Tampilkan hasilnya di `console`.  

2. Operasi dan Percabangan
- Deklarasikan dua variabel `a = 10` dan `b = 20`, lalu bandingkan ukurannya.  
- Jika `a` lebih besar dari `b`, cetak "a lebih besar dari b", jika tidak, cetak "a kurang dari atau sama dengan b".  

Array
3. Operasi Array
- Buat sebuah array `numbers = [3, 7, 1, 9, 4]`.  
- Urutkan angka dalam array dari kecil ke besar, lalu cetak hasilnya.  

4. Filter Array
- Buat sebuah array `nums = [1, 2, 3, 4, 5, 6, 7, 8, 9]`.  
- Saring semua angka genap dan cetak array baru yang berisi angka-angka tersebut.  

Fungsi
5. Fungsi Dasar
- Definisikan sebuah fungsi `add(a, b)` yang mengembalikan hasil penjumlahan `a` dan `b`.  
- Panggil `add(5, 7)`, lalu cetak hasilnya di `console`.  
- (Gunakan fungsi `sort` untuk pengurutan).  

6. Fungsi Rekursif
- Definisikan sebuah fungsi rekursif `factorial(n)` untuk menghitung faktorial dari `n` (n!).  
- Contoh: `factorial(5)` harus mengembalikan `120`.  



Perulangan
7. Menghitung Jumlah dengan For Loop
- Buat sebuah array `arr = [2, 4, 6, 8, 10]`.  
- Gunakan perulangan `for` untuk menghitung jumlah semua elemen dalam array, lalu cetak hasilnya.  

8. Countdown dengan While Loop
- Deklarasikan variabel `count = 5`.  
- Gunakan perulangan `while` untuk mencetak angka dari `5, 4, 3, 2, 1, 0`.  



Objek (Dictionary)
9. Operasi pada Objek
- Buat sebuah objek `person = { name: "Alice", age: 25, city: "Taipei" }`.  
- Tambahkan `1` ke dalam nilai `age` lalu cetak objek `person` yang telah diperbarui.  

10. Iterasi Objek
- Buat sebuah objek `scores = { Alice: 85, Bob: 92, Charlie: 78 }`.  
- Gunakan perulangan `for...in` untuk menelusuri objek `scores`, lalu cetak informasi dalam format seperti `"Alice memiliki skor 85"`.
